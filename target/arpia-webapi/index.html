Users

Resource that manages the processes related to users of the platform.

/users get post

get

Get a information list about a users.

post

Add a new user in the database.

get /users

Get a information list about a users.

Secured by jwt

Arpia REST supports Json Web Token for authenticating all API requests.

  • Request
  • Response
  • Security

Headers

  • Accept-Language: (string)

    It is used to specify the language preferred by the user to receive the error message, by default the API uses en-US.

Query Parameters

  • start: (integer)

    Start query param used to change initial user of the list.

  • size: (integer)

    If this query param is not provides then the service automatically assigns the value 20.

  • type: (one of SUPER_USER, ADMINISTRATOR, OPERATOR, MESSENGER, CLIENT)

    This query param is user type and it's used to filter the list.

Bad JWT request. Unfortunately, re-authenticating the user won't help here.
